# ドラマ感想共有アプリ

---

## 【このアプリについて】<br>
- 見たドラマのあらすじや感想などを投稿、視聴予定のドラマの日付を確認・登録できるアプリ<br>

---

## 【アプリ作成背景】<br>
自分が見たドラマや映画の感想などを他の人も見れ、自分も新しいドラマや映画作品を知ることができたらと思い、このアプリを作成しました<br>

## 【機能一覧】
<ドラマ感想ページ>
- 投稿<br>
- 投稿編集<br>
- 投稿削除<br>
- ページネーション<br>
- 国カテゴリごとの投稿表示<br>

<ドラマカレンダーページ><br>
- カレンダー表示<br>

---

## 【アプリ説明】
<ドラマ感想ページ><br/>
1.「投稿」ボタンクリック<br>
2.投稿ページにて「国名」[^1]、「タイトル」、「感想」、「画像」を入力し、投稿<br>
3.投稿ページにて投稿内容表示<br>

[^1]：プルダウンにて国名を選択
